# בונה תוכן חודשי - מוזה | הוראות התקנה ל-Netlify

## מבנה הפרויקט
```
netlify.toml              קובץ הגדרות
public/
  index.html               דף הכלי שהלקוח פותח
  logo.png                 לוגו מוזה
netlify/
  functions/
    generate.js            פונקציית השרת שמחזיקה את מפתח ה-API
```

## שלב 1: קבלת מפתח API מ-Anthropic
1. כניסה לכתובת console.anthropic.com ופתיחת חשבון (נפרד מהמנוי הרגיל שלך ל-Claude).
2. בתפריט "API Keys", יצירת מפתח חדש והעתקה שלו (מוצג פעם אחת בלבד).
3. טעינת קרדיט לחשבון (אפשר להתחיל בסכום קטן, השימוש בכלי הזה זול מאוד).

## שלב 2: פתיחת חשבון Netlify
1. כניסה לכתובת app.netlify.com והרשמה (אפשר עם חשבון Google/GitHub).
2. אין צורך בכרטיס אשראי לתוכנית החינמית.

## שלב 3: העלאת הפרויקט
**הדרך הפשוטה ביותר, בלי GitHub:**
1. בעמוד הראשי של Netlify יש אזור שכתוב בו "Deploy manually" עם ריבוע לגרירת קבצים.
2. גוררים לתוכו את **כל תיקיית הפרויקט** (לא רק את index.html, את כל התיקייה עם netlify.toml, public, netlify).
3. Netlify יבנה את האתר אוטומטית תוך כמה שניות ויתן לינק זמני.

## שלב 4: הזנת מפתח ה-API באתר
1. בתוך האתר שנוצר ב-Netlify: Site configuration → Environment variables.
2. הוספת משתנה חדש: מפתח בשם `ANTHROPIC_API_KEY`, ערך = המפתח שהעתקת משלב 1.
3. חשוב: אחרי הוספת המשתנה צריך לעשות Deploy מחדש (Deploys → Trigger deploy) כדי שהשינוי ייכנס לתוקף.

## שלב 5: בדיקה
1. פתיחת הלינק של האתר.
2. מילוי השאלון עד הסוף ובדיקה שלוח התוכן נבנה בהצלחה.
3. אם מופיעה שגיאה, לבדוק קודם כל שהמשתנה ANTHROPIC_API_KEY הוזן נכון ושנעשה Deploy מחדש אחריו.

## שלב 6 (רצוי): שם דומיין נוח
Netlify נותן כתובת אקראית כמו `xyz123.netlify.app`. אפשר לשנות אותה תחת Site configuration → Domain management → Options → Edit site name, לכתובת כמו `moza-content.netlify.app`. חיבור דומיין אישי (moza.co.il וכו') אפשרי גם כן דרך אותו מסך.

## עלות שוטפת
אין עלות קבועה. משלמים רק לפי שימוש מול Anthropic (מטבע = טוקנים), וכל פוסט עולה שברירי אגורות. תוכנית ה-Hosting החינמית של Netlify מספיקה בנוחות לשימוש של כלי כזה מול לקוחות.
